Ferramenta FontAwesome => Ferramenta que nos fornece a utilização de ícones 

padding/margin: baixo direita cima esquerda; (relógio)

Posso desenvolver um outro documento CSS e criar fontes 

Exemplo:

:root {
    --font-dm-sans: 'DM Sans', sans-serif;
}

OBS: 'root' é a raiz do projeto
OBS: Para carregar documentos que possuam variáveis CSS, importamos esses antes de qualquer outro documento
